Neste trabalho tentei simular uma loja de eletrônicos, com venda, cadastro de cliente, cadastro de vendedor, cadastro de produtos..

As classes que possuem cadastro e venda, possuem o crud feito e funcionando.
A venda atualiza o valor a cada novo produto inserido já calculado de acordo com a quantidade inserida.

fiz o trabalho totalmente baseado na concessionária desenvolvida durante as aulas e como maior desafio tive a implementação da venda para aceitar mais de um produto por venda.
